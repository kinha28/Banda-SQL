# 🎸 Atualização - Site da Banda Som Que Liberta

## 🇧🇷 Português

Nessa atualização do site da banda *Som Que Liberta*, desenvolvi uma nova página de fotos da banda 📸.

Utilizei um carrossel de imagens com legendas dinâmicas, onde cada foto possui o nome do integrante que aparece na imagem. Também usei JavaScript para criar as animações do carrossel e das legendas, deixando a navegação mais interativa e moderna.

Além disso, trabalhei na organização visual da página para melhorar a experiência do usuário e deixar o layout mais agradável e responsivo.

🔗 Projeto: https://kinha28.github.io/Banda-SQL/

---

## 🇺🇸 English

In this update of the *Som Que Liberta* band website, I developed a new photo gallery page 📸.

I used an image carousel with dynamic captions, where each photo displays the name of the band member featured in the image. I also used JavaScript to create the carousel and caption animations, making the navigation more interactive and modern.

In addition, I worked on the visual organization of the page to improve the user experience and create a cleaner, more responsive layout.

🔗 Project: https://kinha28.github.io/Banda-SQL/
